CSS-grid og Flexbox-layout

Websiden er bygget opp av en del ulike elementer som strekker seg både horisontalt(kolonner) og vertikalt(rader). Siden består derfor av en content div som funker som en wrapper for alt innholdet på siden. Siden innholdet og alle div-ene som wrapper innholdet strekker seg over flere dimensjoner altså både rader og kolonner med videre rader og kolonner i, så er den ytterste div-en (content) basert på CSS-grid layout. Senere ble div-ene som wrapper bildene og tekstene basert på CSS-flexbox layout. Dette er fordi disse er bare på en dimensjons plan, altså bare kolonner eller bare rader og det ble enklere å opprettholde en flytende layout med en flexbox-layout for hver wrapper-div og CSS-grid layout over det hele grunnet grid-gaps osv.

Bruk av HTML canvas og SVG

Canvas er et HTML element som brukes til å tegne grafikk via Javascript. Canvas er brukt ved å først og fremst sette inn en canvas-tag i HTML dokumentet med en spesifisert høyde og bredde. Det er så laget en JavaScript-fil kalt interact.js som inneholder funksjoner som tegner canvas figuren. Her henter man ut canvas elementet fra HTML-dokumentet og i denne sammenheng setter konteksten til å være 2D. Her er det da tegnet ellipser, rektangler og sirkler ved hjelp av definerte metoder for canvas.

SVG står for Scalar Vector Graphics og brukes for å lage grafikk på web. SVG er brukt ved å også ha en svg-tag i HTML dokumentet med definert størrelse i taggen. Her kan man derimot lage tegninger ved å bruke definerte tagger som for eks. ellipse og rect tagger inni SVG taggen. Dette gjør at når man skal lage interaktivitet i form av fargeskifte, figurer som flytter på seg eller endrer størrelse osv. Kan man direkte aksessere de enkelte elementene ved selektorer eller id-er. I canvas har man etter man har laget figurene ved JavaScript funksjoner ikke noe god mulighet for å direkte få tak i enkelte elementer eller støtte av event handlers slik som SVG.

Bruk av jQuery.

JQuery er et Javascript bibliotek som er laget for å gjøre det enklere å bruke JavaScript på ulike nettsider. jQuery biblioteket forenkler bl. a. HTML manipulasjon, event handling og animasjon. Her er det brukt jQuery for å lage interaktiviteten på de ulike bildene. Måten dette gjøres på er at jQuery lar deg direkte aksessere selektorer og elementer ved id eller class ved å bare bruke $(«tag») inne i en $(document).ready(function(){ }); som sørger for at hele HTML dokumentet er ferdig lastet før man kan gjøre noe.

På SVG-en er dette veldig enkelt å gjøre da man kan ha id-er på alle elementene og aksessere de gjennom jQuery ved metoden ovenfor for å så legge på events som click, mouseenter og mouseleave. Ved disse eventsene kan man så legge inn actions for hva som skal skje og her er det brukt blant annet css funksjonen for å endre farge, animate funksjonen for å endre størrelse og show og hide funksjonene. Ved canvas så er det som sagt ikke noen enkel metode for å direkte aksessere de individuelle elementene så her er det bare brukt events som skjer på hele canvaset.

Cross-browser testing

Her er det gjennom utviklingen av websiden testet på to nettlesere (Google Chrome og Safari) ved å kjøre siden på disse for å sørge for at alle funksjoner som blir brukt støttes av nettleserne. Dette ble gjort ved å prøve ut alle nye funsksjoner i begge nettleserne for å se om de var kompatible og fungerte med lik oppførsel i begge nettleserne. Deretter videreutvikles disse og så repeteres testen.
